VJET JavaScript Overview

VJET JavaScript provides a full featured Eclipse IDE for JavaScript, which includes:

  • VJET JavaScript Perspective
  • Enforced Typing
  • Authoring Assistance
  • JavaScript execution and debug support

Begin with the Getting Started Tutorial for an introduction to VJET JS. For those of you not already familiar with Eclipse, please refer to the Eclipse documentation for an overview of the Eclipse Workbench.
