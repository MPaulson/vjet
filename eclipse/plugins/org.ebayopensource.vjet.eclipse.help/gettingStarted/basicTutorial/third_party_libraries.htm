Using VJET JS Type Library Projects

Most developers will want to use some of the popular JavaScript libraries available on the market. VJET JS has created type libraries for a few of the popular libraries, which are available on the VJET JS download site. VJET JS Type Libraries exist as Eclipse projects and can be imported into your workspace.

Before you can import these libraries into Eclipse, download them to an accessible location.

  • Node.js
  • jQuery
  • Dojo
  • JSON

To use a third party or custom library, you must:

  1. Have access to the type library project
  2. Import the type library into Eclipse
  3. Add the type library to your project's build path.

Importing VJET JS Type Libraries

  1. VJET JS Type Libraries exist as Eclipse projects, so the libraries must be imported into your Eclipse workspace. Select File > Import.

  2. Select General > Existing Projects into Workspace > Next.

  3. If you are importing one of the VJET JS libraries select Select archive file:, then Browse.

    Select the type library .zip file you want to import, then select Open.

  4. Select the library project you want to import and Finish.

  5. The imported type library is imported into your workspace and shows up in the Script Explorer.

Add the Type Library to the Project Build Path

In order to reference the libraries in your project, you must add the type library to your project's build path.

  1. Right click on your project and select Build Path > Configure Build Path

  2. Select the Projects tab to view the list of projects you can add to your build path. Press Add and select the projects you want to add.

  3. Select OK > Finish.

  4. The library is added to your project's build path.

  5. Now your project can reference the types in the library using code assist and other authoring features.
