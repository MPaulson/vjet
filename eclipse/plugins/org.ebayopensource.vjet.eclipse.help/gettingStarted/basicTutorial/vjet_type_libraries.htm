VJET JS Type Libraries

VJET JS Type Libraries provide the JavaScript metadata descriptions used to support the VJET JS authoring and type safety features.

The default set of libraries associated with a VJET JS project support popular JavaScript usage.

  • JS Native Types - Java primitives and native types (eg., int, Date)
  • Browser SDK - browser and DOM types (eg., Window, HTMLElement)
  • VJO LIB - VJET JS types

Expand the folders to browse the libraries and to get a sense of the scope of support provided by VJET JS.
